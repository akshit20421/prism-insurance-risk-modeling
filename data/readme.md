# Data Instructions

This project uses the French Motor Third Party Liability Claims dataset.

Due to GitHub file size limits, the raw data is not stored in this repository.

## Download the Data

Download the following files and place them in this folder:

- freMTPL2freq.csv  
- freMTPL2sev.csv  

Search online for:

**"French Motor Third Party Liability Claims dataset"**

## Expected Folder Structure

data/
├── freMTPL2freq.csv
└── freMTPL2sev.csv
